# Çok Dilli Sözlük: İplik Boyama Terminolojisi

Bu sözlük, uluslararası operasyonlarda kullanılan temel terimlerin İngilizce (EN), Almanca (DE), Japonca (JA) ve Türkçe (TR) karşılıklarını içerir.

| English (EN) | Deutsch (DE) | Japanese (JA) | Türkçe (TR) |
| :--- | :--- | :--- | :--- |
| Yarn Dyeing | Garnfärberei | 糸染め (Itozome) | İplik Boyama |
| Color Fastness | Farbechtheit | 染色堅牢度 (Senshoku Kenroudo) | Renk Haslığı |
| Liquor Ratio | Flottenverhältnis | 浴比 (Yokubi) | Flotte Oranı |
| Exhaustion | Aufzug | 固着 (Kochaku/Exhaust) | Egzozyon / Çektirme |
| Fixation | Fixierung | 固着 (Kochaku/Fixation) | Fiksaj |
| Levelness | Egalität | 均一性 (Kin'itsusei) | Egalite / Düzgünlük |
| Bleaching | Bleichen | 漂白 (Hyouhaku) | Ağartma / Kasar |
| Scouring | Abkochen | 精練 (Seiren) | Pişirme / Yıkama |
| Auxiliaries | Hilfsmittel | 助剤 (Jozai) | Yardımcı Kimyasallar |
| Reactive Dye | Reaktivfarbstoff | 反応染料 (Hannou Senryou) | Reaktif Boya |
| Mercerization | Merzerisation | シルケット加工 (Shiruketto) | Merserizasyon |
| Softener | Weichmacher | 柔軟剤 (Juunanzai) | Yumuşatıcı |

| Hank Dyeing | Strangfärberei | 枷染め (Kasezome) | Çile Boyama |
